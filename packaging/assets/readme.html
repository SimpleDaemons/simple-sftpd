simple-sftpd - Read Me

Quick Start

After installation, simple-sftpd will be available as a system service.

Configuration

Edit the configuration file:

/etc/simple-sftpd/simple-sftpd.conf

Service Management

Start the service:

sudo systemctl start simple-sftpd

Enable at boot:

sudo systemctl enable simple-sftpd
