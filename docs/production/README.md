## Overview

The Production Version of Simple Secure FTP Daemon is a complete, production-ready FTP server designed for small to medium deployments.

## What's Included

- ✅ Complete FTP protocol implementation (RFC 959)
- ✅ FTPS support (SSL/TLS)
- ✅ Basic security features (authentication, ACLs)
- ✅ Multi-format configuration (JSON, YAML, INI)
- ✅ Hot reload configuration
- ✅ Cross-platform support
